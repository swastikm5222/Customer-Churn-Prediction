# Customer Churn Prediction

## Problem Statement

Predict customers likely to churn and identify the factors contributing to churn.

## Business Objective

Help businesses proactively retain customers by identifying high-risk users.

## Project Goals

* Perform exploratory data analysis
* Build churn prediction models
* Explain predictions using explainable AI
* Generate retention recommendations
* Deploy an interactive application

## Planned Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* SHAP
* Streamlit
* GitHub
